About the Converter

What is WIPO SEQUENCE SOCKS?

WIPO SEQUENCE SOCKS is a free online tool that converts Excel spreadsheets containing sequence data into WIPO ST.26 compliant XML files. It is designed for patent attorneys and researchers who need to prepare sequence listings for patent applications.

Is this tool free?

Yes, it is completely free to use. No registration or login is required.

Is my data safe?

Yes. Uploaded files are processed in memory only and are not stored on the server. Once the conversion is complete and you download the XML file, all data is automatically discarded. See our Privacy Policy for details.

How many sequences can I convert at once?

The converter can handle 1,000+ sequences in a single Excel file. For best performance, we recommend keeping the file size under 10 MB. If you have more than 5,000 sequences, consider splitting them into multiple files.

About WIPO ST.26

What is WIPO ST.26?

WIPO Standard ST.26 is the international standard for representing nucleotide and amino acid sequence listings in patent applications. It replaced the previous ST.25 standard and uses XML format instead of plain text.

When did ST.26 become mandatory?

WIPO ST.26 became the mandatory format for sequence listings in international patent applications filed on or after July 1, 2022. Most national patent offices have also adopted this standard.

Do I still need WIPO Sequence software?

Yes. Even after using this tool to convert your Excel file to XML, you still need to import the XML into WIPO Sequence for validation and final adjustments before filing. See the installation guide for details.

Troubleshooting

The conversion is taking too long

For large files (500+ sequences), conversion may take 30 seconds to 1 minute. If it takes longer than 2 minutes, try the following:

  • Refresh the page and try again
  • Split the file into smaller batches
  • Check your internet connection

I see a "Server error" message

This usually occurs when the Excel file format is not recognized. Please check:

  • Use the official template downloaded from the home page
  • Save the file in .xlsx format (.xls or .csv are not supported)
  • Make sure you have not renamed or deleted the sheet tabs in the template

"Cannot connect to server"

The conversion server may be temporarily unavailable. Please wait a moment and try again. If the problem persists, let us know via the error report form.

I cannot import the XML into WIPO Sequence

Please check the following:

  • Use the latest version of WIPO Sequence
  • Use "Import Sequence Listing" instead of "Open"
  • Verify that the original Excel data follows the correct format (see Excel Template Tips)
