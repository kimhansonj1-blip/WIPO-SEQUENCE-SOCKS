Instalar WIPO Sequence

Descargue y configure la aplicacion de escritorio oficial

Guias › Instalar WIPO Sequence

¿Que es WIPO Sequence?

WIPO Sequence es la aplicacion de escritorio oficial proporcionada por la OMPI (Organizacion Mundial de la Propiedad Intelectual) para crear y validar listados de secuencias de nucleotidos y aminoacidos en formato XML ST.26. Es necesaria para las solicitudes de patentes que incluyen secuencias biologicas.

Requisitos del sistema

  • Windows: Windows 10 o posterior (64 bits)
  • macOS: macOS 10.14 (Mojave) o posterior
  • Linux: Ubuntu 18.04 o posterior
  • RAM: Minimo 4 GB (8 GB recomendado para listados de secuencias grandes)
  • Disco: Al menos 500 MB de espacio libre

Paso 1: Descargar

  1. Visite la pagina oficial de WIPO Sequence en wipo.int
  2. Navegue a la seccion "Descargar"
  3. Seleccione su sistema operativo (Windows / macOS / Linux)
  4. Descargue el archivo de instalacion
Consejo: Descargue siempre desde el sitio web oficial de la OMPI para asegurarse de tener la version mas reciente con todos los parches de seguridad.

Paso 2: Instalar

Windows

  1. Ejecute el instalador .exe descargado
  2. Siga las instrucciones del asistente de instalacion
  3. Elija el directorio de instalacion (se recomienda el predeterminado)
  4. Haga clic en "Instalar" y espere a que se complete
  5. Inicie WIPO Sequence desde el menu Inicio

macOS

  1. Abra el archivo .dmg descargado
  2. Arrastre el icono de WIPO Sequence a la carpeta Aplicaciones
  3. En el primer inicio, haga clic derecho y seleccione "Abrir" (para omitir Gatekeeper)
  4. Haga clic en "Abrir" en el cuadro de dialogo de seguridad
Nota: En macOS, es posible que aparezca una advertencia de seguridad en el primer inicio. Esto es normal para las aplicaciones descargadas fuera del App Store.

Paso 3: Primer inicio

  1. Abra WIPO Sequence
  2. La aplicacion descargara los datos de referencia necesarios en el primer uso
  3. Espere a que se complete la configuracion inicial (esto puede tardar unos minutos)
  4. Ya esta listo para crear o importar listados de secuencias

Importar XML desde WIPO SEQUENCE SOCKS

Despues de convertir su archivo Excel a XML con nuestra herramienta:

  1. Abra WIPO Sequence
  2. Haga clic en Importar listado de secuencias
  3. Haga clic en Cargar y seleccione su archivo .xml descargado
  4. Introduzca un nombre de proyecto y haga clic en Importar
Consejo: Despues de importar, utilice la validacion integrada de WIPO Sequence para verificar si hay errores antes de enviar su solicitud de patente.
