Installer WIPO Sequence

Telechargez et configurez l'application de bureau officielle

Guides › Installer WIPO Sequence

Qu'est-ce que WIPO Sequence ?

WIPO Sequence est l'application de bureau officielle fournie par l'OMPI (Organisation Mondiale de la Propriete Intellectuelle) pour creer et valider des listages de sequences nucleotidiques et d'acides amines au format XML ST.26. Elle est requise pour les demandes de brevet incluant des sequences biologiques.

Configuration requise

  • Windows : Windows 10 ou ulterieur (64 bits)
  • macOS : macOS 10.14 (Mojave) ou ulterieur
  • Linux : Ubuntu 18.04 ou ulterieur
  • RAM : Minimum 4 Go (8 Go recommande pour les grands listages de sequences)
  • Disque : Au moins 500 Mo d'espace libre

Etape 1 : Telecharger

  1. Visitez la page officielle de WIPO Sequence sur wipo.int
  2. Accedez a la section "Telecharger"
  3. Selectionnez votre systeme d'exploitation (Windows / macOS / Linux)
  4. Telechargez le fichier d'installation
Conseil : Telechargez toujours depuis le site officiel de l'OMPI pour vous assurer d'avoir la derniere version avec tous les correctifs de securite.

Etape 2 : Installer

Windows

  1. Executez le programme d'installation .exe telecharge
  2. Suivez les instructions de l'assistant d'installation
  3. Choisissez le repertoire d'installation (le choix par defaut est recommande)
  4. Cliquez sur "Installer" et attendez la fin de l'installation
  5. Lancez WIPO Sequence depuis le menu Demarrer

macOS

  1. Ouvrez le fichier .dmg telecharge
  2. Glissez l'icone WIPO Sequence dans le dossier Applications
  3. Au premier lancement, faites un clic droit et selectionnez "Ouvrir" (pour contourner Gatekeeper)
  4. Cliquez sur "Ouvrir" dans la boite de dialogue de securite
Remarque : Sur macOS, un avertissement de securite peut apparaitre au premier lancement. C'est normal pour les applications telechargees en dehors de l'App Store.

Etape 3 : Premier lancement

  1. Ouvrez WIPO Sequence
  2. L'application telechargera les donnees de reference necessaires au premier demarrage
  3. Attendez la fin de la configuration initiale (cela peut prendre quelques minutes)
  4. Vous etes pret a creer ou importer des listages de sequences

Importer un XML depuis WIPO SEQUENCE SOCKS

Apres avoir converti votre fichier Excel en XML avec notre outil :

  1. Ouvrez WIPO Sequence
  2. Cliquez sur Importer un listage de sequences
  3. Cliquez sur Charger et selectionnez votre fichier .xml telecharge
  4. Saisissez un nom de projet et cliquez sur Importer
Conseil : Apres l'importation, utilisez la validation integree de WIPO Sequence pour verifier les erreurs avant de soumettre votre demande de brevet.
