About the Converter

What is WIPO SEQUENCE SOCKS?

WIPO SEQUENCE SOCKS is a free online tool that converts Excel spreadsheets containing sequence data into WIPO ST.26 compliant XML files. It is designed for patent attorneys and researchers who need to prepare sequence listings for patent applications.

Is this tool free?

Yes, it is completely free to use. No registration or login is required.

Is my data safe?

Yes. Your uploaded files are processed in memory and never stored on our servers. Once the conversion is complete and you download the XML file, all data is automatically discarded. See our Privacy Policy for details.

How many sequences can I convert at once?

Our converter handles over 1,000 sequences in a single Excel file. For optimal performance, we recommend keeping files under 10 MB. If you have more than 5,000 sequences, consider splitting them into multiple files.

About WIPO ST.26

What is WIPO ST.26?

WIPO Standard ST.26 is the international standard for presenting nucleotide and amino acid sequence listings in patent applications. It replaced the older ST.25 standard and uses XML format instead of plain text.

When did ST.26 become mandatory?

WIPO ST.26 became the mandatory format for sequence listings in international patent applications filed on or after July 1, 2022. Most national patent offices have also adopted this standard.

Do I still need WIPO Sequence software?

Yes. After converting your Excel file to XML using our tool, you should import the XML into WIPO Sequence to validate it and make any final adjustments before submitting your patent application. See our installation guide.

Troubleshooting

The conversion is taking too long

For large files (500+ sequences), conversion may take 30 seconds to 1 minute. If it takes longer than 2 minutes, try:

  • Refreshing the page and trying again
  • Reducing the file size by splitting into smaller batches
  • Checking your internet connection

"Server error" message

This usually means the Excel file format is not recognized. Make sure you:

  • Use the official template (download from the home page)
  • Save the file as .xlsx format (not .xls or .csv)
  • Do not rename or remove the template's sheet names

"Cannot connect to server"

Our conversion server may be temporarily unavailable. Please wait a few minutes and try again. If the problem persists, please report it via the Error Report form.

The XML file won't import into WIPO Sequence

Check the following:

  • Ensure you're using the latest version of WIPO Sequence
  • Use "Import Sequence Listing" (not "Open")
  • Verify that your original Excel data follows the correct format (see Excel Template Tips)
Still having issues? Use the Error Report form to let us know. Include a description of the problem and, if possible, a screenshot.
