Install WIPO Sequence

Download and set up the official desktop application

Guides › Install WIPO Sequence

What is WIPO Sequence?

WIPO Sequence is the official desktop application provided by WIPO (World Intellectual Property Organization) for creating and validating nucleotide and amino acid sequence listings in ST.26 XML format. It is required for patent applications that include biological sequences.

System Requirements

  • Windows: Windows 10 or later (64-bit)
  • macOS: macOS 10.14 (Mojave) or later
  • Linux: Ubuntu 18.04 or later
  • RAM: Minimum 4 GB (8 GB recommended for large sequence listings)
  • Disk: At least 500 MB free space

Step 1: Download

  1. Visit the official WIPO Sequence page at wipo.int
  2. Navigate to the "Download" section
  3. Select your operating system (Windows / macOS / Linux)
  4. Download the installer file
Tip: Always download from the official WIPO website to ensure you have the latest version with all security patches.

Step 2: Install

Windows

  1. Run the downloaded .exe installer
  2. Follow the installation wizard prompts
  3. Choose the installation directory (default is recommended)
  4. Click "Install" and wait for completion
  5. Launch WIPO Sequence from the Start menu

macOS

  1. Open the downloaded .dmg file
  2. Drag the WIPO Sequence icon to the Applications folder
  3. On first launch, right-click and select "Open" (to bypass Gatekeeper)
  4. Click "Open" in the security dialog
Note: On macOS, you may see a security warning on first launch. This is normal for applications downloaded outside the App Store.

Step 3: First Launch

  1. Open WIPO Sequence
  2. The application will download required reference data on first run
  3. Wait for the initial setup to complete (this may take a few minutes)
  4. You're ready to create or import sequence listings

Importing XML from WIPO SEQUENCE SOCKS

After converting your Excel file to XML using our tool:

  1. Open WIPO Sequence
  2. Click Import Sequence Listing
  3. Click Upload and select your downloaded .xml file
  4. Enter a Project Name and click Import
Tip: After importing, use WIPO Sequence's built-in validation to check for any errors before submitting your patent application.
