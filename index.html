WIPO SEQUENCE SOCKS

HOW TO USE
  • 1. Push DOWNLOAD! to get Template.xlsx
  • 2. Complete the template file (see examples)
  • 3. Push UPLOAD! and select the written file
  • 4. Push SUBMIT! to get converted .xml file
  • (It might take about 30 sec ~ 1 min)
  • 5. Open WIPO Squence program and Click 'IMPORT SEQUENCE LISTING'
  • 6. Click Upload and select the converted .xml file
  • 7. Fill out the Project Name and Push 'Import'
  • 8. Done!
