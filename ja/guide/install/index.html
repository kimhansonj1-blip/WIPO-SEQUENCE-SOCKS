WIPO Sequence インストール

公式デスクトップアプリケーションのダウンロードとセットアップ

ガイド › WIPO Sequence インストール

WIPO Sequenceとは？

WIPO Sequenceは、WIPO（世界知的所有権機関）が提供する公式デスクトップアプリケーションで、ST.26 XML形式で核酸・アミノ酸配列表を作成・検証するために使用します。生物学的配列を含む特許出願に必須のツールです。

動作環境

  • Windows：Windows 10以降（64ビット）
  • macOS：macOS 10.14（Mojave）以降
  • Linux：Ubuntu 18.04以降
  • RAM：最小4GB（大きな配列表には8GB推奨）
  • ディスク：500MB以上の空き容量

ステップ1：ダウンロード

  1. 公式WIPOサイト wipo.int にアクセスする
  2. 「ダウンロード」セクションに移動する
  3. お使いのOS（Windows / macOS / Linux）を選択する
  4. インストーラーファイルをダウンロードする
ヒント：最新バージョンとすべてのセキュリティパッチが適用されていることを確認するため、必ずWIPO公式サイトからダウンロードしてください。

ステップ2：インストール

Windows

  1. ダウンロードした .exe インストーラーを実行する
  2. インストールウィザードの指示に従う
  3. インストール先ディレクトリを選択する（デフォルトを推奨）
  4. 「インストール」をクリックして完了まで待つ
  5. スタートメニューからWIPO Sequenceを起動する

macOS

  1. ダウンロードした .dmg ファイルを開く
  2. WIPO SequenceアイコンをApplicationsフォルダにドラッグする
  3. 初回起動時は右クリックして「開く」を選択する（Gatekeeperを回避するため）
  4. セキュリティダイアログで「開く」をクリックする
注意：macOSでは、初回起動時にセキュリティ警告が表示される場合があります。これはApp Store外からダウンロードしたアプリケーションでは通常の動作です。

ステップ3：初回起動

  1. WIPO Sequenceを開く
  2. 初回実行時にアプリケーションが必要な参照データをダウンロードする
  3. 初期セットアップが完了するまで待つ（数分かかる場合があります）
  4. 配列表の作成またはインポートの準備が完了しました

WIPO SEQUENCE SOCKSからのXMLインポート

本ツールでExcelファイルをXMLに変換した後：

  1. WIPO Sequenceを開く
  2. Import Sequence Listing（配列表インポート）をクリックする
  3. Upload（アップロード）をクリックし、ダウンロードした .xml ファイルを選択する
  4. プロジェクト名を入力し、Import（インポート）をクリックする
ヒント：インポート後は、特許出願を提出する前にWIPO Sequenceの組み込み検証機能を使ってエラーがないか確認してください。
