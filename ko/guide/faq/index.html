변환기에 대하여

WIPO SEQUENCE SOCKS란 무엇인가요?

WIPO SEQUENCE SOCKS는 서열 데이터가 포함된 엑셀 스프레드시트를 WIPO ST.26 호환 XML 파일로 변환해주는 무료 온라인 도구입니다. 특허 출원을 위한 서열목록을 준비해야 하는 변리사 및 연구자를 위해 설계되었습니다.

이 도구는 무료인가요?

네, 완전히 무료로 사용할 수 있습니다. 회원가입이나 로그인이 필요하지 않습니다.

제 데이터는 안전한가요?

네. 업로드된 파일은 메모리에서만 처리되며, 서버에 저장되지 않습니다. 변환이 완료되고 XML 파일을 다운로드하면 모든 데이터는 자동으로 삭제됩니다. 자세한 내용은 개인정보처리방침을 참조하세요.

한 번에 몇 개의 서열을 변환할 수 있나요?

본 변환기는 단일 엑셀 파일에서 1,000개 이상의 서열을 처리할 수 있습니다. 최적의 성능을 위해 파일 크기를 10MB 이하로 유지하는 것을 권장합니다. 서열이 5,000개를 초과하는 경우 여러 파일로 분할하는 것을 고려하세요.

WIPO ST.26에 대하여

WIPO ST.26이란 무엇인가요?

WIPO 표준 ST.26은 특허 출원 시 뉴클레오타이드 및 아미노산 서열목록을 표현하기 위한 국제 표준입니다. 기존의 ST.25 표준을 대체하였으며, 일반 텍스트 대신 XML 형식을 사용합니다.

ST.26은 언제부터 의무화되었나요?

WIPO ST.26은 2022년 7월 1일 이후 출원된 국제 특허 출원의 서열목록에 대한 필수 형식이 되었습니다. 대부분의 국가 특허청도 이 표준을 채택하였습니다.

WIPO Sequence 소프트웨어가 여전히 필요한가요?

네. 본 도구를 사용하여 엑셀 파일을 XML로 변환한 후에도, 특허 출원 전에 WIPO Sequence에 XML을 가져와 검증하고 최종 조정을 해야 합니다. 설치 가이드를 참조하세요.

문제 해결

변환이 너무 오래 걸립니다

대용량 파일(500개 이상의 서열)의 경우 변환에 30초~1분이 소요될 수 있습니다. 2분이 넘게 걸리는 경우 다음을 시도해 보세요:

  • 페이지를 새로 고침한 후 다시 시도
  • 파일을 더 작은 단위로 분할하여 크기 줄이기
  • 인터넷 연결 상태 확인

"서버 오류" 메시지가 표시됩니다

이는 보통 엑셀 파일 형식이 인식되지 않는 경우 발생합니다. 다음 사항을 확인하세요:

  • 홈 페이지에서 다운로드한 공식 템플릿 사용
  • 파일을 .xlsx 형식으로 저장 (.xls 또는 .csv 불가)
  • 템플릿의 시트 이름을 변경하거나 삭제하지 않았는지 확인

"서버에 연결할 수 없습니다"

변환 서버가 일시적으로 사용할 수 없는 상태일 수 있습니다. 잠시 기다린 후 다시 시도해 주세요. 문제가 지속되면 오류 보고 양식을 통해 알려주세요.

XML 파일을 WIPO Sequence에서 가져올 수 없습니다

다음 사항을 확인하세요:

  • 최신 버전의 WIPO Sequence 사용
  • "Open" 대신 "Import Sequence Listing" 사용
  • 원본 엑셀 데이터가 올바른 형식을 따르는지 확인 (엑셀 템플릿 작성법 참조)
여전히 문제가 있으신가요? 오류 보고 양식을 사용하여 알려주세요. 문제에 대한 설명과 가능하다면 스크린샷을 첨부해 주세요.
