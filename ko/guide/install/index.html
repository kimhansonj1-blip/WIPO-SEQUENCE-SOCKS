WIPO Sequence란?

WIPO Sequence는 WIPO(세계지식재산기구)가 제공하는 공식 데스크탑 애플리케이션으로, ST.26 XML 형식의 뉴클레오타이드 및 아미노산 서열목록을 작성하고 검증하는 데 사용됩니다. 생물학적 서열이 포함된 특허 출원 시 반드시 필요합니다.

시스템 요구사항

  • Windows: Windows 10 이상 (64비트)
  • macOS: macOS 10.14 (Mojave) 이상
  • Linux: Ubuntu 18.04 이상
  • RAM: 최소 4GB (대용량 서열목록의 경우 8GB 권장)
  • 디스크: 500MB 이상의 여유 공간

1단계: 다운로드

  1. 공식 WIPO Sequence 페이지 wipo.int에 접속합니다
  2. "Download" 섹션으로 이동합니다
  3. 운영체제(Windows / macOS / Linux)를 선택합니다
  4. 설치 파일을 다운로드합니다
팁: 항상 공식 WIPO 웹사이트에서 다운로드하여 최신 버전과 모든 보안 패치가 포함된 버전을 사용하세요.

2단계: 설치

Windows

  1. 다운로드한 .exe 설치 파일을 실행합니다
  2. 설치 마법사의 안내에 따라 진행합니다
  3. 설치 디렉토리를 선택합니다 (기본값 권장)
  4. "Install"을 클릭하고 완료될 때까지 기다립니다
  5. 시작 메뉴에서 WIPO Sequence를 실행합니다

macOS

  1. 다운로드한 .dmg 파일을 엽니다
  2. WIPO Sequence 아이콘을 Applications 폴더로 드래그합니다
  3. 최초 실행 시 마우스 오른쪽 버튼으로 클릭 후 "열기"를 선택합니다 (Gatekeeper 우회)
  4. 보안 대화상자에서 "열기"를 클릭합니다
참고: macOS에서는 최초 실행 시 보안 경고가 표시될 수 있습니다. App Store 외부에서 다운로드한 애플리케이션의 경우 정상적인 현상입니다.

3단계: 최초 실행

  1. WIPO Sequence를 엽니다
  2. 최초 실행 시 필요한 참조 데이터를 자동으로 다운로드합니다
  3. 초기 설정이 완료될 때까지 기다립니다 (몇 분 소요될 수 있습니다)
  4. 이제 서열목록을 작성하거나 가져올 준비가 되었습니다

WIPO SEQUENCE SOCKS에서 XML 가져오기

본 도구를 사용하여 엑셀 파일을 XML로 변환한 후:

  1. WIPO Sequence를 엽니다
  2. Import Sequence Listing을 클릭합니다
  3. Upload를 클릭하고 다운로드한 .xml 파일을 선택합니다
  4. Project Name을 입력하고 Import를 클릭합니다
팁: 가져온 후에는 특허 출원 전에 WIPO Sequence의 내장 검증 기능을 사용하여 오류 여부를 확인하세요.
