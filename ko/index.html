WIPO SEQUENCE SOCKS

엑셀 파일로 WIPO SEQUENCE에 서열을 입력하는 도구

한국어 | ENGLISH

사용 방법

1
템플릿 다운로드을 클릭하여 엑셀 템플릿 파일을 받으세요.
2
엑셀 템플릿 파일 내 서열정보를 작성하세요.
(예시 참조)
3
파일 선택을 클릭하여 작성한 엑셀 파일을 첨부하세요.
4
변환을 클릭하여 XML 형식의 파일을 받으세요.
(30초 ~ 1분 소요)
5
WIPO Sequence 프로그램을 열어 Import Sequence Listing를 클릭하세요.
6
Upload를 클릭하여 조금 전 받은 xml 파일을 첨부 및 Project Name 작성 → Import를 클릭하세요.
1단계

엑셀 파일 준비

엑셀 템플릿 파일을 받은 후 파일 내 서열정보를 작성하세요.

다운로드 (한국어) 다운로드 (영어)
3, 4단계

XML 파일로 변환

작성한 엑셀 파일을 XML 형식의 파일로 변환하세요.
(30초 ~ 1분 소요될 수 있음)
