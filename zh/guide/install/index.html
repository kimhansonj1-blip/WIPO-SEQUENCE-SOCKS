安装 WIPO Sequence

下载并设置官方桌面应用程序

指南 › 安装 WIPO Sequence

什么是 WIPO Sequence？

WIPO Sequence 是 WIPO（世界知识产权组织）提供的官方桌面应用程序，用于以 ST.26 XML 格式创建和验证核苷酸及氨基酸序列表。凡包含生物序列的专利申请均需使用此工具。

系统要求

  • Windows：Windows 10 或更高版本（64位）
  • macOS：macOS 10.14（Mojave）或更高版本
  • Linux：Ubuntu 18.04 或更高版本
  • 内存：最低 4 GB（大型序列表建议 8 GB）
  • 磁盘：至少 500 MB 可用空间

第一步：下载

  1. 访问 WIPO 官方网站 wipo.int
  2. 进入"下载"部分
  3. 选择您的操作系统（Windows / macOS / Linux）
  4. 下载安装程序文件
提示：请务必从 WIPO 官方网站下载，以确保您获得包含所有安全补丁的最新版本。

第二步：安装

Windows

  1. 运行下载的 .exe 安装程序
  2. 按照安装向导的提示操作
  3. 选择安装目录（建议使用默认路径）
  4. 点击"安装"并等待完成
  5. 从开始菜单启动 WIPO Sequence

macOS

  1. 打开下载的 .dmg 文件
  2. 将 WIPO Sequence 图标拖到"应用程序"文件夹
  3. 首次启动时，右键点击并选择"打开"（以绕过 Gatekeeper）
  4. 在安全对话框中点击"打开"
注意：在 macOS 上，首次启动时可能会出现安全警告。对于从 App Store 以外下载的应用程序，这是正常现象。

第三步：首次启动

  1. 打开 WIPO Sequence
  2. 应用程序将在首次运行时下载所需的参考数据
  3. 等待初始设置完成（可能需要几分钟）
  4. 现在可以创建或导入序列表了

从 WIPO SEQUENCE SOCKS 导入 XML

使用本工具将 Excel 文件转换为 XML 后：

  1. 打开 WIPO Sequence
  2. 点击 Import Sequence Listing（导入序列表）
  3. 点击 Upload（上传）并选择您下载的 .xml 文件
  4. 输入项目名称并点击 Import（导入）
提示：导入后，在提交专利申请之前，请使用 WIPO Sequence 的内置验证功能检查是否存在任何错误。
